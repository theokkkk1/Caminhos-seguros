# Caminhos Seguros

Jogo educativo desenvolvido em Python utilizando a biblioteca Pygame.

O objetivo do jogo é conscientizar sobre mobilidade urbana, segurança para pedestres e desafios encontrados nas cidades.

---

# Tecnologias Utilizadas

- Python 3.10+
- Pygame

---

# Como Clonar o Repositório

Abra o terminal e execute:

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
````

Depois entre na pasta do projeto:

```bash
cd Caminhos-seguros-main
```

---

# Como Instalar as Dependências

## 1. Criar Ambiente Virtual (Opcional, mas recomendado)

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 2. Instalar as Bibliotecas

```bash
pip install -r requirements.txt
```

---

# Como Executar o Projeto

Execute:

```bash
python main.py
```
